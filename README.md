# PromptPilot

### Professional Offline AI Prompt Workspace for ComfyUI

PromptPilot is a modern desktop application designed to help creators build high quality prompts for image and video generation completely offline.

Built specifically for ComfyUI users, PromptPilot works with Ollama to generate detailed prompts without sending your ideas or data to the cloud.

---

## Why PromptPilot?

Most prompt tools are web based or require online AI services.

PromptPilot was created with one goal:

**Keep everything on your own computer.**

Your prompts stay local.

Your images stay local.

Your models stay local.

No subscriptions.

No cloud accounts.

No API keys required.

# Installation

## Windows

Download the latest release.

Extract the archive.

Run:

PromptPilot.exe

---

## Linux

Extract the archive.

Run:

install_desktop_linux.sh

Launch PromptPilot from your desktop or applications menu.
